python基础知识总结（二）

Destinyu ♥ 6 February 2020 python 3 min


函数

定义函数

  • 无参数
  def function():
    print("this is a function")
  • 传一个参数
  def function(name):
    print("hello,"+name)

传递多参数

  • 位置参数（由位置关系关联）
  def position_function(a,b):
    print(a+" need "+b)
  
  position_function('Tom','Jerry')#根据参数的位置决定输出
  • 关键字实参
  def key_function(a,b):
    print(a+" need "+b)
  
  key_function(a='Tom',b='Jerry')#直接由等式决定，可不管顺序
  • 默认值（使用默认值时必须先列出没有默认值的形参）
  def default_function(a,b='Jerry'):
    print(a+" need "+b)
  
  default_function('Tom')
  default_function(a='Tom')

返回值

  • 返回简单值
  def function(a,b):
    meaasge=a+' '+b)
    return message
  
  function_back=function('Tom','Jerry')
  print(function)
  • 实参可选
  def function(a,b,c=''):
    if c:
        print(a+" need "+b+" and "+c)
    else:
        print(a+" need "+b)
          
  function_back=function('Tom','Jerry')
  function_back=function('Tom','Jerry','Mike')
  • 返回字典
  def function(a,b):
    name={'first':a,'last':b}
    return name
  
  function_back=function('Tom','Jerry')

传递列表

  • 把参数看做列表
  def function(values):
    for value in values:
        print(value)
          
  names=['a','b','c']
  function(names)
  • 不修改原列表，创建副本
  def function(values[:])

传递任意数量的参数

  • 任意数量实参
  def function(*names):
    print(names)
  
  function('a')
  function('a','b','c')
  • 结合位置实参和任意数量参数
  def function(age,*names):
    print(str(age)+" years old:")
    for name in names:
        print(name)
        
  function(12,'a')
  function(13,'a','b','c')
  • 使用任意数量的关键字实参
  def function(first,last,**info):
    dictionary={}
    dictionary['first_name']=first
    dictionary['last_name']=last
    for key,value in info.items():
        dictionary[key]=value
    return dictionary
    
  user=function('a','b',age=12,address='China')
  print(user)

函数存入模块

  • 导入整个函数
  import function
  • 导入某模块的特定函数
  from module_name import function
  • 给函数指定别名
  from module_name import function as fn
  • 给模块指定别名
  import module_name as mn
  • 导入模块中的所有函数
  from module_name import *

创建和使用类

导入类

文件和异常

从文件中读取数据

  • 读取整个文件
  with open('a.txt') as file_object:#文件放在同一文件夹
    contents=file_object.read()
    print(contents)
  • 逐行读取
  with open('a.txt') as file_object:
    for line in file_object:
        print(line.rstrip())
  • 创建包含文件各行内容的列表
  with open('a.txt') as file_object:
    lines=file_object.readlines()
    
  for line in lines:
    print(line.rstrip())

写入文件

  • 写入空文件
  with open('b.txt','w') as file_object:
    file_object.write("I love programming")
  • 写入多行
  with open('b.txt','w') as file_object:
    file_object.write("I love programming\n")
    file_object.write("I love DL\n")
  • 附加到文件
  with open('c.txt','a') as file_object:
    file_object.write("I love programming")

  try:

  • try-except-else

  • 保存和读取用户生成数据
  import json
  filename='username.json'
  try:
    with open(filename) as f_obj:
        username=json.load(f_obj)
  except FileNotFoundError:
    username=input("What is your name?")
    with open(filename,'w') as f_obj:
        json.dump(username,f_obj)
        print("we will remeber you when you back,"+username)
  else:
    print("welcome back,"+username+"!")
  • 重构（将代码重新划分为一系列具体工作的函数）
  import json
  def get_stored_username():
    filename='username.json'
    try:
        with open(filename) as f_obj:
            username=json.load(f_obj)
    except FileNotFoundError:
        return None
      else:
        return username
  
  def get_new_username():
    username=input("What is your name?")
    filename='username.json'
    with open(filename,'w') as f_obj:
        json.dump(username,f_obj)
    return username
  
  def greet_user():
    username=get_stored_username()
    if username:
        print("Wellcome back,"+username)
    else:
        username=get_new_username()
        print("we'll remeber when you come back, "+username)
  
  greet_user()
