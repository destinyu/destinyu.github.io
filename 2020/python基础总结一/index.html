python基础知识总结(一)

Destinyu ♥ 4 February 2020 python 2 min

变量与简单数据类型

变量名规范

  • 应使用小写字母
  • 字母之间可以用下划线连接
  • 便于阅读但不能过于省略

字符串操作

  • 单引号和双引号均可，根据实际包含情况来决定

  • 转换大小写的用法

  string.title()#首字母大写
   string.upper()#全部大写
   string.lower()#全部小写

  • 合并字符串：+

  • 添加空白与换行：\t+\n

  • 删除多余空白

  string.rstrip()#删除末尾空白
   string.lstrip()#删除开头空白
   string.strip()#删除字符串两端空白

数字

  • 乘方：a**b

  • 类型转换：

  str(number)
  • 特别注意：

python2：

3⁄2=1

3.0/2=1.5

3⁄2.0=1.5

3.0/2.0=1.5

python3：

3⁄2=1.5

python之禅

  • Complex is better than complicated
  • Simple is better than complex
  • Now is better than never

列表的应用

列表及其索引

  • 索引从0开始
  list=['a','b','c','d']
   print(list[0])#从零开始
   print(list[3])

列表的修改操作

  • 修改：
  list[0]='e'
  • 添加：
  list.append('f')
   list.append(0,'f')#位置插入
  • 删除：
  del list[0]#无法再次访问
  
  list.pop()#可再次访问，删最后一块
  list.pop(0)#弹出任何位置的元素
  poped_list#删除的存储到这里
  
  list.remove('a')#删除指定值元素

组织列表

  • 直接排序
  list.sort()#由小到大
  list.sort(reverse=True)#反向
  • 使用排序结果但不改变原序列
  sorted(list)
   sorted(list,reverse=True)#反转
  • 反转列表元素
  list.reverse()
  • 列表长度
  len(list)



操作列表

列表遍历

  • for循环(注意缩进)
  for list in lists:
    print(list)

数值列表

  • range
  for value in range(1,5):#不会打印5
    print(value)
  • 数字列表
  numbers=list(range(1,6))
  even_numbers=list(range(2,11,2))#每次加2
  • 对数字列表进行统计
  min(list)#最大值
  max(list)#最小值
  sum(list)#总和
  • 用for循环生成数字列表
  squares=[]
  for value in range(1,11):
    square=value**2
    squares.append(square)
  • 使用列表解析直接生成
  squares=[value**2 for value in range(1,11)]

使用列表的一部分

  • 切片
  list[1:4]#提取2-4的元素
  list[:4]#从表头开始
  list[-3:]#从表尾倒数三个
  • 复制列表
  new_list=list[:]#复制列表的副本

元组

  • 定义一个元组
  dimensions=(200,50)#不能改值
  • 重新赋值
  dimensions=(300,50)



if语句

一条if语句

  • 区分大小写

  • 注意缩进

  • 可以使用and和or判断两个条件

  • 检查特定值是否包含在列表：

  if value in list
  if value not in list

if组合句

  • if-else
  • if-elif-else
  • if-elif…-else

字典

字典及其使用

  • 创建
  dictionary={'a':'b','c':'d'}
  • 访问
  dictionary['a']
  dictionary['c']
  • 添加键值对
  dictionary['e']='f'
  • 删除
  del dictionary['a']

遍历字典

  • 遍历所有键值对
  for key,value in dictionary.items():
    print(key)
    print(value)
  • 遍历所有键
  for key in dictionary.keys()
  for key in dictionary#可省略
  for key in sorted(dictionary.keys())#排序后的结果
  • 遍历所有值
  for value in dictionary.values()
  for value in set(dictionary.values())#去掉重复值

嵌套

  • 字典列表
  d1={'a':1}
  d2={'b':2}
  d3={'c':3}
  list=[d1,d2,d3]
  • 在字典中存列表
  dictionary={
    'a':1,
    'b':[1,2]
  }
  • 在字典中套字典
  dictionary={
    'a'={
        'first':1,
        'last':2,
    },
    'b'={
        'first':3,
        'last':4,
    }
  }



用户输入和while循环

input

  • 使用int来获取数值
  number=input("Please input your number: ")
  num=int(number)

while

  • 简单的while循环
  current_number=1
  while current_number<=5:
    print(current_number)
    current_number+=1
  • 使用判断来退出
  message=""
  while message!='quit':
    message=input("please input any message")
    if message!='quit':
        print message
  • 使用标志
  active = True
  while active:
    message = input("...")
    if message == 'quit':
        active=False
      else:
        print(message)
  • break和continue
  while True:
    message = input("...")
    if message == 'quit'
        break#直接跳出循环，结束
      else:
        print message

  while True:
    message = input("...")
    if message == 'quit'
        continue#不理会。返回循环开头继续
      else:
        print message

用while来处理字典和列表

  • 列表之间元素移动
  unconfirmed=['a','b','c']
  confirmed=[]
  while unconfirmed:
    current=unconfirmed.pop()
    confirmed.append(current)
  • 删除列表中特定元素
  list=['a','b','c','d']
  while 'b' in list:
    list.remove('b')
