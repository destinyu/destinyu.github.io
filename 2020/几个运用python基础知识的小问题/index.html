Destinyu ♥ 17 February 2020 python 1 min

斐波那契数列

思路

前两个数都是1，第三个数开始每个数都是前两个数之和

ps：在leetcode中只需要编写方法

方法

class Solution():
    
    def fib(self,N):
        if N==0:
            return 0
        elif N==1:
            return 1
        else:
            return self.fib(N - 1) + self.fib(N - 2)

num=input("输入:")
out=Solution()
print(out.fib(int(num)))

完美数

思路

所有除自身以外的因子的和恰好等于它本身

方法

  • 比较好想的方法：遍历所有数，凡是可以被除开的，都存入列表里面，然后将列表相加得到结果，与原数比较
  class Solution_1():#重复判断一次
      def checkPerfectNumber(self,x):
  
          a=[]
          num=0
          sum=0
          for factor in range(int(x),1,-1):
              if int(x)%factor==0:
                  a.append(int(x)//factor)
                  sum+=a[num]
                  num += 1
          if sum==int(x):
               return True
          else:
              return False

问题：存在一次重复判断，并且用列表复杂度高，超时


  • 标准解法：为了不超时，判断到开平方即可。1和小于1的数都不是完美数。从2开始将因子都相加。
  class Solution_2():
      def checkPerfectNumber(self,num):
          if num<=1:
              return False
          sum=1
          for i in range(2,int(sqrt(num)+1)):
              if num%i==0:
                  sum+=i+num//i
          if sum==num:
              return True

​

  • 另解：看到一种方法，是想从2开始判断，用i<num/i来控制因子大小，不会重复判断两次。

细看这个i<num/i应该就是sqrt(num)的来源


  • 巧解：因为完美数一共就5个，可以直接输出
